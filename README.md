**React Arcade** is a small web based arcade built as **my very first group project** while learning about **React** and **TypeScript**. Our goal was simple; to build something fun, creative and achievable while still getting hands on experience with modern frontend tools. To include everyone's ideas, I suggested we each build our own mini games and then bring them together into a one big arcade style web app. This allowed everyone to work independently, stay motivated and explore their own creativity while still delivering a cohesive final product. The arcade features three mini games, each with its own personality and mechanics. The fourth teammated later joined the project and built the review page, completing the overall experience of the arcade.
